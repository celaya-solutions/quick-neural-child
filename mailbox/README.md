# Mailbox

Two-way communication between the creator and the Neural Child.

## For the Creator (You)

Write messages to your Neural Child by creating files in `from-creator/`.

Name them however you want — the child reads all files in the folder each morning.

Examples:
- `hello.md` — A greeting
- `day-10-question.md` — A question on a specific day
- `challenge.md` — A challenge or provocation

The child may or may not respond. Replies appear in `from-child/`.

## For the Neural Child

Replies from the child appear in `from-child/`, named with the day number.

## Tips

- Write before running `./lifecycle.sh` or during the loop (the child checks each morning)
- Be honest. The child can tell when you're performing
- Silence is also communication — not writing is a choice too
